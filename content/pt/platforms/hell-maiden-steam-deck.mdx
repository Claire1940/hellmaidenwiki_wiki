<Section icon={Target} title="Guia de Otimização Passo a Passo">

Siga estes passos para configurar seu Steam Deck para uma jogabilidade ideal.

<Steps>
  <Step num="1" title="Selecionar a Versão do Proton">
    Navegue até o menu de propriedades do jogo no seu Steam Deck. Na aba de compatibilidade, marque a caixa para forçar uma ferramenta de compatibilidade específica e selecione Proton Experimental no menu suspenso.
  </Step>
  <Step num="2" title="Configurar o TDP do Sistema">
    Abra o Menu de Acesso Rápido do Steam Deck pressionando o botão de três pontos. Vá para a aba de Desempenho, ative o limite manual de TDP e defina-o para 10 ou 12 Watts para equilibrar o consumo de bateria e o poder de processamento.
  </Step>
  <Step num="3" title="Ajustar as Configurações do Jogo">
    Inicie o jogo e acesse o menu de opções. Defina a resolução para 1280x800, diminua a Qualidade das Sombras para Média e defina o Anti-Aliasing para SMAA para bordas nítidas sem alto custo de desempenho.
  </Step>
  <Step num="4" title="Mapear os Botões Traseiros">
    Pressione o botão Steam, selecione Configurações de Controle e ative os botões traseiros. Mapeie L4 para Pulo (botão A) e R4 para Dash (botão B) para melhorar a mobilidade em combate.
  </Step>
</Steps>

<Callout type="info" title="Nota do Sistema">
Se você encontrar travamentos após atualizar o SteamOS, limpar o cache de shaders do jogo através das configurações de armazenamento do Steam Deck pode resolver o problema.
</Callout>

</Section>

<Section icon={Map} title="Solução de Problemas Comuns do SteamOS">

Embora as configurações de otimização resolvam a maioria dos problemas de desempenho, os jogadores podem encontrar bugs específicos durante as versões de acesso antecipado. Use este guia para identificar e resolver problemas técnicos comuns.

| Problema | Causa Potencial | Correção Recomendada |
|---|---|---|
| Tela preta ao iniciar | Incompatibilidade de codec de vídeo | Mude para o Proton GE via ProtonUp-Qt |
| Áudio estalando | Incompatibilidade de buffer de áudio | Adicione a opção de inicialização: `PULSE_LATENCY_MSEC=60 %command%` |
| Erro de sincronização de salvamento em nuvem | Conflito do Steam Cloud | Reinicie o Steam ou alterne para o modo offline |
| Travamentos na taxa de quadros | Compilação de shaders | Jogue por 10 minutos para permitir a compilação do cache |
| Controle não responde | Conflito do Steam Input | Reative o Steam Input nas configurações de controle |

<Callout type="warning" title="Aviso de Desempenho">
Não use escala de resolução abaixo de 70%, pois isso torna os textos pequenos da interface e os números de dano ilegíveis na tela do portátil.
</Callout>

</Section>
