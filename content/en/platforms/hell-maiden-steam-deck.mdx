<Section icon={Target} title="Step-by-Step Optimization Guide">

Follow these steps to configure your Steam Deck for optimal gameplay.

<Steps>
  <Step num="1" title="Select Proton Version">
    Navigate to the game properties menu on your Steam Deck. Under the compatibility tab, check the box to force a specific compatibility tool and select Proton Experimental from the drop-down menu.
  </Step>
  <Step num="2" title="Configure System TDP">
    Open the Steam Deck Quick Access Menu by pressing the three-dot button. Go to the Performance tab, enable manual TDP limit, and set it to 10 or 12 Watts to balance battery consumption and processing power.
  </Step>
  <Step num="3" title="Adjust In-Game Settings">
    Launch the game and access the options menu. Set the resolution to 1280x800, lower Shadow Quality to Medium, and set Anti-Aliasing to SMAA for clear edges without high performance costs.
  </Step>
  <Step num="4" title="Map Back Paddles">
    Press the Steam button, select Controller Settings, and enable the back buttons. Map L4 to Jump (A button) and R4 to Dash (B button) for improved combat mobility.
  </Step>
</Steps>

<Callout type="info" title="System Note">
If you encounter stuttering after updating SteamOS, clearing the shader cache for the game via Steam Deck storage settings can resolve the issue.
</Callout>

<Section icon={Map} title="Troubleshooting Common SteamOS Issues">

While optimization settings resolve most performance issues, players may encounter specific bugs during early access builds. Use this guide to identify and resolve common technical issues.

| Issue | Potential Cause | Recommended Fix |
|---|---|---|
| Black screen on startup | Video codec incompatibility | Switch to Proton GE via ProtonUp-Qt |
| Audio crackling | Audio buffer mismatch | Add launch option: `PULSE_LATENCY_MSEC=60 %command%` |
| Cloud save sync error | Steam Cloud conflict | Restart Steam or toggle offline mode |
| Frame rate stuttering | Shader compilation | Play for 10 minutes to allow cache compilation |
| Controller unresponsive | Steam Input conflict | Re-enable Steam Input in controller settings |
