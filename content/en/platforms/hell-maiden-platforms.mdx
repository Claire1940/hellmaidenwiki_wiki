<Section icon={Sparkles} title="PC System Requirements & Hardware Analysis">

Because the game forces you to confront massive waves of blood-thirsty demons simultaneously, understanding the hardware requirements is essential for maintaining a smooth frame rate. Rogue-lite horde survival games are notoriously taxing on the CPU due to the sheer volume of enemy pathfinding, projectile tracking, and damage calculations occurring at any given millisecond.

The game is built to run on modern 64-bit operating systems, specifically requiring Windows 10 or newer. Below is the detailed breakdown of both the minimum and recommended system specifications to help you evaluate your current hardware setup.

| Requirement | Minimum Specs | Recommended Specs |
| :--- | :--- | :--- |
| OS | Windows 10 or newer (64-bit) | Windows 10 or newer (64-bit) |
| Processor | 4 Cores / 2.5 GHz+ | 4 Cores / 3.5 GHz+ |
| Memory | 8 GB RAM | 16 GB RAM |
| Graphics | GTX 970 / AMD RX 570 | GTX 1070 / AMD RX 570 |
| Storage | 4 GB available space | 4 GB available space |

While the GPU demands are relatively modest—allowing older cards like the Nvidia GTX 970 to achieve playable frame rates—investing in a faster multi-core processor will yield the most significant performance gains. When utilizing complex card combinations, ultimate attacks, and screen-clearing status effects, a high-frequency CPU prevents the micro-stuttering that often plagues the later stages of a run.

<Callout type="tip" title="Hardware Recommendation">
If you are running the game on a laptop with integrated graphics, such as Intel Iris Xe, ensure your system memory is running in dual-channel mode. This significantly boosts memory bandwidth, which the game relies on to render massive enemy hordes.
</Callout>

<Section icon={Target} title="Step-by-Step Performance Optimization Guide">

If you are experiencing performance drops on older hardware or want to maximize the battery life of your portable device, following a structured optimization process can dramatically improve your gameplay experience. Use these steps to configure your system for the perfect balance of visual clarity and fluid performance.

<Steps>
  <Step num="1" title="Adjust In-Game Video Settings">
    Launch the game and navigate to the options menu. Set the display mode to Fullscreen rather than Borderless Windowed to give the game direct priority over system resources. Turn off V-Sync if you are using a variable refresh rate monitor, and reduce the particle density to medium.
  </Step>
  <Step num="2" title="Configure Steam Launch Options">
    Right-click the game in your Steam Library, select Properties, and find the Launch Options field. Input optimization arguments such as `-high` to force your CPU to prioritize the game's threads, or `-force-d3d11` if you experience stability issues on older graphics cards.
  </Step>
  <Step num="3" title="Optimize Handheld Power Limits">
    On Steam Deck or similar handhelds, open the quick-access performance overlay. Enable a manual GPU clock control limit of 1200MHz and set the TDP limit to 9 Watts. This keeps the processor running at peak efficiency while preventing thermal throttling during intense boss fights.
  </Step>
  <Step num="4" title="Update Graphics Drivers">
    Ensure your graphics drivers are updated to the latest version. Both Nvidia and AMD frequently release hotfixes that optimize memory allocation for indie titles utilizing DirectX 11 and DirectX 12 rendering pipelines.
  </Step>
</Steps>

<Callout type="info" title="Performance Impact">
By capping your frame rate to 60 FPS on handheld devices, you can reduce power consumption by up to 25% while maintaining a completely fluid gameplay experience during high-density combat phases.
</Callout>
