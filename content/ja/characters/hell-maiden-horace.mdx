<Section icon={Shield} title="戦闘戦略：辺獄から愛欲へ">

ホラティウスのキットを効果的に使うには、現在利用可能な2つの圏の間の遷移を理解する必要があります。辺獄では、脅威は主に近接攻撃が中心（インプ、スケルトン）ですが、愛欲ではバイパーやサキュバスのような、より高速で攻撃的な敵に直面します。

| 敵のタイプ | ホラティウスの対抗戦略 | 優先度 |
| :--- | :--- | :--- |
| **インプ / スケルトン** | 標準的なカルペ・ディエム射撃 | 低 |
| **エリートバイパー** | 集中ダメージ + 呪文バースト | 高 |
| **エリートサキュバス** | 横移動 + 発射体連射 | 高 |
| **ミノス女王** | 予兆のある範囲攻撃用にダッシュを温存 | ボス |
| **スカルミリョーネ** | 硬直中にホラティウスの呪文を使用 | ボス |

<Callout type="tip" title="高度なポジショニング">
第2圏：愛欲では、常に脱出路を確保しておいてください。ホラティウスの武器は強力ですが、正確なエイムを必要とします。囲まれてしまうと、**エリートバイパー**のような優先ターゲットへの集中攻撃ができなくなります。
</Callout>

実績**「地獄の熟達 II」**を目指す際は、ホラティウスのアビリティが**クリスタルレアリティのチャーム**と相性が良いことを覚えておいてください。これらのレアアイテム（実績「天の贈り物」）を入手することで、敵の密度がピークに達する愛欲の12分間を生き残るために必要なステータスの底上げが可能になります。

</Section>
