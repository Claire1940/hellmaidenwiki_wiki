<Section icon={Swords} title="不死身の月ビルド：セレニック・ブルーム・メタ">

現在の**ヘル・メイデン最強ビルド**の頂点は「不死身の月」構成です。このビルドは**セレニック・ブルーム**という武器を中心に据えており、魔法の月を放ち、持続的な範囲攻撃（AoE）リングを形成します。基礎ダメージは控えめですが、ヒット率が高いため、ライフスティールや状態異常付与のモディファイアに最適です。中盤までには、敵の群れの中心に立ち止まったまま体力を回復し続けることが可能になります。

| コンポーネント | 推奨カード | 選定理由 |
| :--- | :--- | :--- |
| **核となる武器** | **Selenic Bloom** | 広大なAoEと高い攻撃頻度 |
| **回復Mod** | **Eternal Prestige** | ヒット時に15%の確率で回復（レベル3時） |
| **ダメージMod** | **Heroic Surge** | サイズとダメージの両方を大幅に増加 |
| **ユーティリティMod** | **Projectile Count** | 最大4つの月を同時に出現させる |
| **優先ステータス** | **Attack Size** | 回復ゾーンの直径を拡大 |

このビルドを実行するには、早い段階で**Heroic Surge**に集中する必要があります。攻撃速度は低下しますが、生み出される圧倒的なダメージ増加により、数少ない月でも画面を掃討できます。**Eternal Prestige**がレベル3に達すると、4つの月が数十体の敵にダメージを与え続け、毎分数百回の15%回復チャンスが発生するため、生存能力はほぼ無限になります。

<Callout type="success" title="ビルドの強み">
このビルドはボス戦で真価を発揮します。辺獄の守護者の最終フェーズでは、月が「セーフゾーン」を作り出し、弾幕を避けながらHPを回復することができます。
</Callout>

<Section icon={Sparkles} title="合成とMod戦略">

最適化こそが、5分での敗北と20分での勝利を分ける境界線です。ヘル・メイデンでは単に良いカードを選ぶだけでは不十分です。手札を管理し、戦略的に合成する必要があります。Modカードはレベル3まで進化させることができ、各レベルで基礎ステータスとカードの視覚効果が大幅に強化されます。

**重要な合成ルール:**
- **同レベルのみ**: レベル1を2枚でレベル2に、レベル2を2枚でレベル3に合成できます。
- **レアリティ一致**: 同じレアリティ（枠の色で判別可能）のカードのみ合成可能です。
- **手札管理**: 手札の枠には限りがあります。重複したレベル1カードを持ちすぎると、新しい武器やユーティリティModを入れるスペースがなくなります。

| ステータスカテゴリ | 優先度 | 推奨チャーム |
| :--- | :--- | :--- |
| **序盤** | **高** | **Charm of Growth**（経験値獲得 +28%） |
| **中盤** | **中** | **Magnetic Pole**（アイテム回収距離） |
| **終盤** | **高** | **Elite Slayer**（ボスダメージ +40%） |
| **防御** | **中** | **Charm of Aegis**（被ダメージ軽減 8%） |

<Callout type="tip" title="マグネット戦略">
**Magnetic Pole** Modを軽視しないでください。終盤、画面は紫色の経験値オーブで埋め尽くされます。マグネットがないと、レベルアップのために危険な敵の群れに突っ込まざるを得なくなります。レベル3のマグネットがあれば、安全を確保しながら戦力を強化し続けることができます。
</Callout>

</Section>
