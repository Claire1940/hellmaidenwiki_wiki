地獄の階層で長期生存するためには、遭遇するカードのレアリティと種類を理解することが不可欠です。

| カードタイプ | 主な機能 | 戦略的価値 |
| :--- | :--- | :--- |
| **武器カード** | 基本攻撃/魔法 | ビルドの**核**であり、プレイスタイルを決定する |
| **ブーストカード** | 単一ステータスの上昇 | 特定のダメージスケーリングにおいて**高い**価値 |
| **エフェクトカード** | 状態異常（炎上/鈍足）の付与 | 集団制御（CC）が必要な場合に**中程度**の価値 |
| **スプレッドカード** | 隣接するカードにバフを共有 | 複数武器のシナジーにおいて**非常に優秀** |
| **ミラクルカード** | 複数の強力な強化 | **最高ティア**のレアリティ。常に最優先 |
| **カースカード** | 代償を伴う強力な力 | **状況次第**。ハイリスク・ハイリターン |

<Callout type="tip" title="配置の重要性">
Modカードに方向指定の要件があるか常に確認してください。一部のカードは左右どちらかの武器しか強化しないため、デッキの順序はカードそのものと同じくらい重要です。
</Callout>

<Section icon={Swords} title="「チェインライトニング」スピードビルド">

**チェインライトニング**ビルドは、高い攻撃速度と画面全体の敵を一掃することに焦点を当てています。蛇行する弾丸を発射する**ウェイファインダー (Wayfinder)** シグネチャー武器を使用することで、**アイ・オブ・ラブ (I of Love)** Modカードを安定して発動させることができます。このビルドは、体力の低い敵が大量に出現する「辺獄 (Limbo)」ステージの攻略に最適です。

<CardGrid cols={2}>
  <Card title="必要な武器">
    - **ウェイファインダー**: メインの弾丸供給源。
    - **ライトニングボルト**: パッシブな範囲攻撃。
    - **ブレイジングクイル**: 近距離の防御用。
  </Card>
  <Card title="主要なModシナジー">
    - **アイ・オブ・ラブ**: レベル3で80%の確率でチェインライトニングが発生。
    - **スウィフトストライク**: 攻撃速度のペナルティを相殺。
    - **インビゴレート**: 300%の圧倒的なダメージブースト。
  </Card>
</CardGrid>

このビルドを最大化するには、ダメージと速度のトレードオフを管理する必要があります。**インビゴレート**カードはダメージを200%から300%増加させますが、攻撃速度を大幅に低下させます。画面を埋め尽くす敵を処理するために必要な高頻度のライトニング連鎖を維持するには、**スウィフトストライク**との組み合わせが唯一の方法です。

| ビルド構成要素 | レベル1の効果 | レベル3の効果 |
| :--- | :--- | :--- |
| **アイ・オブ・ラブ発動** | 30%の連鎖確率 | **80%の連鎖確率** |
| **雷ダメージ** | 基本ストライク | **合計ダメージ +300%** |
| **連鎖ターゲット** | 敵6体 | **敵10体以上** |
| **攻撃範囲** | 標準 | **範囲カバー率 +60%** |

<Callout type="warning" title="ビルドの弱点">
このビルドはいわゆる「ガラスの砲台（攻撃特化）」です。回復Modがないため、エリート敵に囲まれないよう、**ダッシュ距離**や**移動速度**のチャームに頼る必要があります。
</Callout>

</Section>

<Section icon={Sparkles} title="「不滅の月」サステインビルド">

地獄の第2階層のボス戦で苦戦しているプレイヤーにとって、**不滅の月**ビルドはすべての**ヘル・メイデンのビルド**の中で最高の選択肢です。これは、星の輪のような持続的な範囲攻撃（AoE）を作成する武器**セレニック・ブルーム (Selenic Bloom)** を中心としています。ライフスティール（吸血）メカニズムと組み合わせることで、最も攻撃的な魔物に対してもその場に踏みとどまって戦うことができます。

<Steps>
  <Step num="1" title="セレニック・ブルームを入手する">
    詩人**ホラティウス (Horace)** を救出することでこの武器をアンロックします。これが回復ゾーンの起点となります。
  </Step>
  <Step num="2" title="エターナル・プレステージをスタックする">
    月に**エターナル・プレステージ (Eternal Prestige)** Modカードを適用します。レベル3では、ヒットごとに15%の確率でHPを吸収します。
  </Step>
  <Step num="3" title="軌道を拡大する">
    **ヒロイックサージ (Heroic Surge)** と**攻撃範囲**Modを使用して、月を大きくします。輪の中にいる敵が多いほど、回復速度が上がります。
  </Step>
  <Step num="4" title="弾数を増やす">
    弾数Modをレベルアップさせて3〜4つの月を同時に放ち、複数の回復ゾーンを重ね合わせます。
  </Step>
</Steps>

| シナジーの組み合わせ | 得られる効果 | 戦闘での実用性 |
| :--- | :--- | :--- |
| **月 + HP再生** | 絶え間ない体力回復 | ボスの攻撃を耐えるのに**非常に優秀** |
| **月 + サイズアップ** | 広大なゾーン制御 | NPCや目標の防衛に**良好** |
| **月 + 弾数増加** | 複数の星の輪 | **高い**DPSと回復頻度 |

<Callout type="success" title="サステインの頂点">
**エターナル・プレステージ**がレベル3に達すると、月の輪で敵の群れを攻撃し続けている限り、軽微な飛び道具のダメージは実質的に無視できるようになります。
</Callout>

</Section>

<Section icon={Shield} title="生存のマイルストーンと進行">

パワーアップのスケーリングは時間との戦いです。**ヘル・メイデンのビルド**が敵の体力増加に遅れないようにするためには、15〜20分のランの間に特定の目標を達成する必要があります。

<Checklist
  id="run-progression"
  title="重要なランのマイルストーン:"
  items={[
    "最初のボス戦までにレベル10に到達する",
    "8分経過までに少なくとも1つのレベル3 Modカードを作成する",
    "第2階層に入る前に全4つの武器スロットを解放する",
    "序盤に少なくとも1つの「成長のチャーム」（経験値増）を装備する",
    "階層ボスを撃破してアルティメットスペルトークンをアンロックする"
  ]}
/>

序盤の焦点は常に**経験値獲得**にあるべきです。オーブからの経験値を14%から28%増加させるチャームは、デッキを完全に最大化するために必要な高レベル（レベル50以上）に到達するために必須です。これらがないと、Modをレベル3に合成するための十分なカードを見つけるのに苦労することになります。

| マイルストーン時間 | 目標レベル | ビルドの状態 |
| :--- | :--- | :--- |
| **0-5 分** | レベル 15 | 武器2種、経験値チャーム発動中 |
| **5-10 分** | レベル 30 | レベル3 Mod 1つ、武器3種 |
| **10-15 分** | レベル 45 | デッキ完成、複数のレベル3 Mod |
| **15分以上** | レベル 55+ | エンドゲームのスケーリング、スペルトークン準備完了 |

<Callout type="tip" title="スペルトークンの管理">
**ダンテのインフェルノ (Dante's Inferno)** やその他のアルティメットスペルは、画面がエリートの「目玉」敵で完全に埋まった時のために取っておきましょう。これらのスペルは一時的な無敵状態と、画面を一掃する圧倒的なダメージを提供します。
</Callout>
