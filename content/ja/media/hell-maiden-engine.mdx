<Section icon={Sparkles} title="ヘル・メイデン・エンジンの習得：デッキのシナジー">

**ヘル・メイデン・エンジン**の核となるのは、デッキ構築インターフェースです。プレイヤーはしばしば、すべてのスロットを高ダメージ武器で埋めてしまう間違いを犯し、プレッシャーの下で崩壊するビルドを作ってしまいます。このエンジンは、武器の速度、発射体数、そしてサポートモッドを考慮したバランスの取れたアプローチに報酬を与えます。

| カードタイプ | 主な機能 | 隣接ボーナス |
| :--- | :--- | :--- |
| **ウェポンカード** | 直接的なダメージ出力 | モッドカードからバフを受ける |
| **スピリットカード** | パッシブなステータス上昇 | 二次的効果を誘発可能 |
| **モッドカード** | ウェポンカードを強化 | 武器に隣接している必要あり |
| **アルティメットカード** | 強力な画面クリア攻撃 | 解放には詩人の救出が必要 |
| **専用武器** | 独自のプレイスタイルの核 | 特定の詩人のパークで強化 |

これらのカードの配置は極めて重要です。一部のモッドカードは、その直前（左）または直後（右）にある武器にしか影響を与えません。強力な「マルチショット」モッドを、互換性のある武器に接していないスロットに配置した場合、そのスロットを事実上無駄にしていることになります。このエンジンは、地獄の円環を進むにつれて、常に手札を再評価するように設計されています。

  </Card>
</CardGrid>

<Section icon={Target} title="詩人救出システム">

ヘル・メイデンにおける進行は、辺獄（リンボ）の詩人たちと密接に結びついています。あなたはただ生き残るために戦っているのではありません。古代の偉大な思想家たちを解放するために戦っているのです。これらのキャラクターを救出することだけが、**ヘル・メイデン・エンジン**における戦術的な選択肢を広げる唯一の方法です。

<Steps>
  <Step num="1" title="詩人を特定する">
    各レベルには、詩人が捕らえられている特定のクエストや隠しエリアがあります。ミニマップに表示される、彼らの場所を示すユニークなアイコンに注目してください。
  </Step>
  <Step num="2" title="チャレンジをクリアする">
    詩人は通常、エリートの「悪鬼」に守られているか、解放するために特定の生存時間を必要とします。救出を開始するには、これらの守護者を倒さなければなりません。
  </Step>
  <Step num="3" title="親交を深め、解放する">
    救出された詩人は拠点に戻ります。彼らと話すことで、詩人一人につき5つの新しいアビリティがアンロックされるほか、ベースステータスを変更するユニークなパークが得られます。
  </Step>
  <Step num="4" title="専用装備を統合する">
    特定の親密度に達すると、詩人は専用武器へのアクセスを許可します。これらはしばしば、高レベルのメタビルドを定義する強力な装備となります。
  </Step>
</Steps>

| 詩人 | 歴史的役割 | 主要なアンロック | プレイスタイルの焦点 |
| :--- | :--- | :--- | :--- |
| **ホメロス** | 叙事詩人 | 伝説の弓 | 遠距離からの精密攻撃 |
| **オウィディウス** | 変身物語 | 変身モッド | 適応的なアビリティ切り替え |
| **ホラティウス** | 風刺詩人 | クリティカル率パーク | ハイリスク・ハイリターン |
| **ルカヌス** | ファルサリア | 出血/DoT効果 | 持続的なプレッシャー |

<Callout type="success" title="アンロックの優先順位">
新規プレイヤーは、まず**ホメロス**を優先的に救出すべきです。彼のアンロック要素は、序盤で最も安定したダメージバフを提供し、第一円環の周回を大幅にスムーズにしてくれます。
</Callout>

<Section icon={Shield} title="サバイバル戦術とボスメカニクス">

**ヘル・メイデン・エンジン**は、現在の早期アクセス状態では地獄の主要な二つの円環を特徴としており、それぞれが過酷なボス戦で締めくくられます。これらのボスは単なる障害物ではありません。彼らはあなたの最も強力なアビリティの源です。ボスを倒すと、彼らの特徴的な力を手に入れ、自分のデッキに加える機会が得られます。

| ボス名 | 円環 | 主な脅威 | 報酬アビリティ |
| :--- | :--- | :--- | :--- |
| **ミノス** | 辺獄 | 画面全域のレーザー | 裁きの光（Judgment Beam） |
| **ケルベロス** | 貪食 | 3レーンの突進攻撃 | 3連撃モッド |
| **エリート悪鬼** | 各地 | 高速の発射体 | 追尾スピリット |

効果的な戦闘には、ダッシュの習得が不可欠です。最も一般的な死因は「ダッシュの枯渇」です。危険がない時に移動を速めるためにダッシュを使ってしまい、ボスが回避不能な攻撃を繰り出した時にクールダウン中になってしまうケースです。ダッシュは移動手段ではなく、防御手段として扱ってください。
