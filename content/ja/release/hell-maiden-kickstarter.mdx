<Section icon={Trophy} title="バッカーティアと限定報酬">

**ヘル・メイデン Kickstarter**では、基本的なデジタル支援からハイエンドな「大悪魔（Arch-Demon）」ティアまで、サポーター向けに様々なエントリーポイントを用意しています。各レベルは価値を提供すると同時に、プロジェクトが最終的な資金調達目標に到達するのを助けるように設計されています。物理的な報酬は特に注目に値し、ゲーム独自の芸術性を反映した高品質のグッズが特徴です。

| ティア名 | 価格 (USD) | 主な報酬 |
| :--- | :--- | :--- |
| **迷える魂** | $15 | デジタル壁紙、クレジットへの名前掲載、Discordロール |
| **メイデン** | $30 | デジタルゲーム本編、デジタルサウンドトラック、ベータアクセス |
| **地獄の学者** | $60 | 物理アートブック、エナメルピンセット、デジタルガイド |
| **業火の騎士** | $120 | コレクターズボックス、サウンドトラックCD、物理ゲームディスク |
| **神聖なる設計者** | $500 | NPCのデザイン、サイン入りポスター、全過去報酬 |

<Callout type="warning" title="数量限定">
「神聖なる設計者」および「高等悪魔（High Demon）」ティアは、開発者がカスタムコンテンツを適切にゲームに統合できるよう、少人数のバッカーに限定されています。
</Callout>

標準的なティアに加え、早期バッカーにはキャンペーン開始後48時間以内に10%の割引が適用されました。この戦略により、プロジェクトは開始からわずか6時間で初期目標金額に到達し、AstralShiftの新しいIPに対する計り知れない関心の高さを証明しました。
