<Section icon={Swords} title="戦闘の核心とカード合成のメカニクス">

**ヘル・メイデン デモ版**の核心には、攻撃能力と防御能力を左右する洗練されたカードシステムがあります。単純なレベルアップの選択肢の代わりに、プレイヤーは特定の攻撃やバフを表すカードを収集します。マスターすべき最も重要なメカニクスは、これらのカードを合成して進化させる能力です。重複したカードや互換性のあるカードを組み合わせることで、威力を高め、二次効果を追加し、最終的には伝説的なスキルへと変貌させることができます。

| メカニクス | 説明 | 戦略的優先度 |
| :--- | :--- | :--- |
| **カード合成** | 2枚の同一カードを組み合わせてレベルとダメージを増加させる | **高** |
| **カード進化** | 最大レベルのカードを新しい強力な形態に変化させる | **極めて重要** |
| **大群サバイバル** | カードが自動発動または起動する間、ポジショニングを維持する | **中** |
| **味方の支援** | 戦闘中にウェルギリウスやオウィディウスのユニークなバフを活用する | **高** |

敵に囲まれているときは、すぐにカードを合成しないでください。クールダウンの遅い1つの高レベル攻撃よりも、2つの別々の低レベル攻撃の方が広範囲をカバーできる場合があります。アップグレードする前に、一瞬の安全な隙を待ちましょう。
</Callout>

<Section icon={Sparkles} title="主要キャラクターと伝説的な味方">

**ヘル・メイデン デモ版**の物語は、象徴的なキャラクターたちによって推進されます。あなたの旅は孤独なものではありません。かつてこの暗い道を歩んだ詩人やガイドたちがあなたを助けてくれます。これらのキャラクターの役割を理解することは、物語愛好家にとっても、ゲームプレイ上の利点を求めるプレイヤーにとっても不可欠です。

| キャラクター | ゲーム内での役割 | 背景リファレンス |
| :--- | :--- | :--- |
| **ダンテ** | プレイ可能な主人公 | 救済を求めて旅をする者 |
| **ウェルギリウス** | ガイド兼味方 | 最後まで同行する忠実な仲間 |
| **オウィディウス** | 伝説的な詩人 | 伝承やバフで支援する辺獄（リンボ）の詩人 |
| **ミノス王妃** | ボス / 敵対者 | 深淵の燃え盛る審判者 |

ウェルギリウスは、アップグレードとナビゲーションの主要な連絡先となります。デモ版では、彼の存在はしばしば安全地帯や主要な敵ウェーブの間の区切りを意味します。一方、オウィディウスはより専門的な支援を提供し、辺獄の階層に現れて他では見られないユニークなカードの選択肢を提示することがよくあります。

<Callout type="info" title="キャラクターとの交流">
**ヘル・メイデン デモ版**は戦闘に焦点を当てていますが、可能な限りウェルギリウスやオウィディウスと会話することで、隠されたカードの相乗効果を解放したり、ミノス王妃の攻撃パターンに関するヒントを得たりすることができます。
</Callout>

<Section icon={Target} title="デモ版ステップバイステップ攻略">

デモ版を進めるには、探索と効率的な戦闘のバランスが必要です。勝利可能なデッキを携えてミノス王妃との最終決戦に到達するために、以下の手順に従ってください。

<Steps>
  <Step num="1" title="最初の降下と基本デッキ">
    辺獄階層の最初のウェーブをクリアすることから始めます。少なくとも3つの異なる攻撃カードを集めることに集中してください。最初の5分間は回復アイテムが稀なため、早めにダメージを受けるのは避けましょう。
  </Step>
  <Step num="2" title="合成の習得">
    最初の重複カードを入手したら、合成システムを使用します。主要な飛び道具攻撃のアップグレードを優先してください。これにより複数の敵を貫通できるようになり、次の「愛欲」階層で不可欠となります。
  </Step>
  <Step num="3" title="ウェルギリウスとの遭遇">
    デモ版の中盤あたりでウェルギリウスに遭遇します。彼は「伝説の恩恵」の選択肢を提示します。ダメージ出力を最大化するために、自分の最高レベルのカードを補完するものを選んでください。
  </Step>
  <Step num="4" title="ミノス王妃の試練">
    デモ版のクライマックスはミノス王妃との戦いです。彼女の「炎の審判」による地上攻撃を避けるために動き続けましょう。AoE（範囲攻撃）カードで召喚された手下を遠ざけつつ、彼女に火力を集中させてください。
  </Step>
</Steps>

<Callout type="warning" title="ボスに関する警告">
ミノス王妃は体力が50%になるとフェーズが変化します。彼女の攻撃速度は大幅に上昇します。このフェーズが始まる前に、少なくとも1つの防御用または移動速度アップ用のカードを有効にしておいてください。
</Callout>

</Section>
