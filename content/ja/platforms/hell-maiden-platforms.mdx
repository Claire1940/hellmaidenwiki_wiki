<Section icon={Sparkles} title="PCシステム要件とハードウェア分析">

本作では血に飢えた悪魔の巨大な群れと同時に戦うことになるため、スムーズなフレームレートを維持するためのハードウェア要件を理解しておくことが不可欠です。ローグライト・ホードサバイバルゲームは、敵の経路探索、発射体の追跡、ダメージ計算がミリ秒単位で大量に発生するため、CPUに大きな負荷がかかることで知られています。

ゲームは最新の64ビットオペレーティングシステムで動作するように構築されており、具体的にはWindows 10以降が必要です。現在のハードウェア構成を確認するために、最小スペックと推奨スペックの詳細は以下の通りです。

| 要件 | 最小スペック | 推奨スペック |
| :--- | :--- | :--- |
| OS | Windows 10以降 (64-bit) | Windows 10以降 (64-bit) |
| プロセッサ | 4コア / 2.5 GHz以上 | 4コア / 3.5 GHz以上 |
| メモリ | 8 GB RAM | 16 GB RAM |
| グラフィックス | GTX 970 / AMD RX 570 | GTX 1070 / AMD RX 570 |
| ストレージ | 4 GB の空き容量 | 4 GB の空き容量 |

GPUへの要求は比較的控えめで、Nvidia GTX 970のような古いカードでもプレイ可能なフレームレートを維持できますが、より高速なマルチコアプロセッサに投資することで、最も顕著なパフォーマンス向上が得られます。複雑なカードの組み合わせ、アルティメット攻撃、画面全体を覆うステータス効果を使用する際、高周波CPUは後半ステージで発生しがちなマイクロスタッター（カクつき）を防いでくれます。

<Callout type="tip" title="ハードウェアの推奨事項">
Intel Iris Xeなどの内蔵グラフィックスを搭載したラップトップでプレイする場合は、システムメモリがデュアルチャネルモードで動作していることを確認してください。これにより、大量の敵を描画するために必要なメモリ帯域幅が大幅に向上します。
</Callout>

</Section>

最高の携帯体験を得るために、3つの重要なパフォーマンスカテゴリにおける動作状況を分析しました。

<Section icon={Target} title="ステップバイステップ・パフォーマンス最適化ガイド">

古いハードウェアでパフォーマンスが低下する場合や、携帯デバイスのバッテリー寿命を最大化したい場合は、以下の最適化プロセスに従うことで、ゲームプレイ体験を劇的に改善できます。

<Steps>
  <Step num="1" title="ゲーム内のビデオ設定を調整する">
    ゲームを起動し、オプションメニューに移動します。ディスプレイモードを「ボーダーレスウィンドウ」ではなく「フルスクリーン」に設定し、ゲームにシステムリソースを直接優先させます。可変リフレッシュレートモニターを使用している場合はV-Syncをオフにし、パーティクル密度を「中」に下げてください。
  </Step>
  <Step num="2" title="Steamの起動オプションを設定する">
    Steamライブラリでゲームを右クリックし、「プロパティ」を選択して「起動オプション」フィールドを見つけます。CPUにゲームのスレッドを優先させる `-high` や、古いグラフィックスカードで安定性の問題がある場合は `-force-d3d11` などの最適化引数を入力します。
  </Step>
  <Step num="3" title="ハンドヘルドの電力制限を最適化する">
    Steam Deckや同様のハンドヘルドデバイスでは、クイックアクセス・パフォーマンスオーバーレイを開きます。手動GPUクロック制御を1200MHzに制限し、TDP制限を9ワットに設定します。これにより、激しいボス戦中のサーマルスロットリングを防ぎつつ、プロセッサをピーク効率で動作させることができます。
  </Step>
  <Step num="4" title="グラフィックスドライバーを更新する">
    グラフィックスドライバーが最新バージョンであることを確認してください。NvidiaとAMDの両社は、DirectX 11およびDirectX 12レンダリングパイプラインを使用するインディータイトルのメモリ割り当てを最適化するホットフィックスを頻繁にリリースしています。
  </Step>
</Steps>

<Callout type="info" title="パフォーマンスへの影響">
ハンドヘルドデバイスでフレームレートを60 FPSに制限することで、高密度の戦闘フェーズでも完全に流動的なゲームプレイを維持しながら、消費電力を最大25%削減できます。
</Callout>

</Section>
