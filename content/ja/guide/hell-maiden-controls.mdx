<Callout type="info" title="クイックガイド">
- **ヘル・メイデンの操作**は、自動攻撃と手動照準、そしてアクティブな回避を組み合わせた、戦術的なサバイバル体験を提供します。

<Section icon={Keyboard} title="キーボードとマウスの入力レイアウト">

**ヘル・メイデンの操作方法**を理解することは、辺獄（Limbo）の深淵を生き延びるための第一歩です。近接度に基づいて攻撃が完全に自動化されている多くの伝統的なホードサバイバルゲームとは異なり、本作ではプレイヤーが積極的に武器を脅威に向ける必要があります。キーボードとマウスの設定におけるこの「ツインスティック」的なアプローチは、「Blazing Quill」や「Winged Lyric」のような集中ダメージビルドにおいて最高の精度を発揮します。

| キー | アクション | 説明 |
| :--- | :--- | :--- |
| **W, A, S, D** | 移動 | 円環（Circle）内を8方向に移動します。 |
| **マウスカーソル** | 照準 | すべての自動攻撃をポインターの方向に向けます。 |
| **スペースキー** | ダッシュ | 一時的な無敵状態を伴う、素早い移動。 |
| **E / 右クリック** | スペル使用 | 固有の詩人スペル（例：ダンテのインフェルノ）を発動します。 |
| **Tab / I** | デッキを開く | 現在のスピリットカードとモッドカードを確認・整理します。 |
| **Esc** | ポーズメニュー | 設定へのアクセス、実績の確認、またはランの終了。 |

<Callout type="tip" title="エイム（照準）戦略">
ダンテは自動的に攻撃するため、プレイヤーの主な焦点は「カイト（引き撃ち）」にあります。マウスカーソルを敵の群れの最も密集している部分に向け続けながら、安全な距離を保つためにダンテを反対方向に移動させましょう。
</Callout>

<Section icon={Gamepad2} title="コントローラーとゲームパッドの設定">

よりリラックスした、あるいは「弾幕シューティング」スタイルの体験を好むプレイヤーにとって、ゲームパッドでの**ヘル・メイデンの操作**は非常に直感的です。ゲームは標準的なツインスティックシューターのレイアウトを採用しています。これは、周囲のモブを一掃するために常に射撃方向を回転させる必要がある「Projectile Pressure（弾幕プレッシャー）」ビルドにおいて特に効果的です。

| ボタン (Xbox/PS) | アクション | 優先度 |
| :--- | :--- | :--- |
| **左スティック** | キャラクターの移動 | 高 |
| **右スティック** | 照準の方向 | 高 |
| **RB / R1 / B** | ダッシュ / 回避 | 最優先 |
| **LB / L1 / A** | スペルの発動 | 中 |
| **View / Select** | デッキ管理 | 中 |
| **Menu / Start** | ゲームの一時停止 | 低 |

<Callout type="warning" title="コントローラーの照準">
右スティックは常にいずれかの方向に倒しておくようにしてください。スティックがニュートラル位置に戻ると、ダンテが特定のターゲットに対して正確に射撃しなくなる可能性があり、12分間のウェーブピーク時には致命的となる場合があります。
</Callout>

<Section icon={Target} title="戦闘と移動メカニズムのマスター">

ヘル・メイデンでの生存は、移動と攻撃的な照準をいかにうまくバランスさせるかにかかっています。辺獄（Limbo）や愛欲（Lust）といった円環における12分間のタイマーは、単に円を描いて歩くだけでは対処できないほど密集したウェーブをもたらします。**ヘル・メイデンの操作**をマスターするということは、ダッシュのリズムと詩人スペルのタイミングを学ぶことを意味します。

<Steps>
  <Step num="1" title="防御的ダッシュ">
    ダッシュは最も重要な生存ツールです。これにより、ダンテは敵や発射物をダメージを受けずに通り抜けることができます。単に速く移動するためだけに最後のダッシュチャージを使わないでください。「包囲された」状態からの脱出や、クイーン・ミノスの範囲攻撃（AoE）を避けるために、常に1回分は残しておきましょう。
  </Step>
  <Step num="2" title="手動照準の優先順位">
    エリート・バイパーやサキュバスのような優先ターゲットを最初に倒すべきです。移動（WASD/左スティック）で群れを避けつつ、マウスや右スティックを使ってこれらの特定の敵を追跡してください。
  </Step>
  <Step num="3" title="神殿の起動">
    移動中、神殿に遭遇することがあります。これらを起動するには、短時間その付近に留まる必要があります。移動操作を使って神殿の周りを「周回」し、進行状況バーを溜めつつ、敵を円の外側に保つようにしましょう。
  </Step>
</Steps>

<Callout type="success" title="高度な戦術">
ボスを「通り抜ける」ダッシュが可能です。もしクイーン・ミノスにアリーナの端へ追い詰められた場合、発射物の壁を迂回しようとするよりも、彼女のモデルを直接ダッシュで突き抜ける方が安全なことが多いです。
</Callout>

| メカニズム | 入力タイプ | ゲームプレイへの影響 |
| :--- | :--- | :--- |
| **ダッシュの無敵時間** | アクティブ | アニメーション中、すべての被ダメージを無効化します。 |
| **自動射撃** | パッシブ | ターゲットが範囲内/照準内にいる限り、継続的なDPSを与えます。 |
| **スペルバースト** | アクティブ | 画面全体の敵を一掃します。移動が制限されている時に使用しましょう。 |
| **カードの並べ替え** | メニュー | 位置によるモッドボーナスを通じてダメージを最適化します。 |

</Section>

<Section icon={Settings} title="インターフェースとデッキ管理の操作">

**ヘル・メイデンの操作**におけるユニークな側面は、ランの最中にスピリットカードの「手札」を並べ替えられることです。これは単なる整理のためではありません。多くのモッドカードは、その位置に基づいてボーナスを提供します（例：「右隣の武器のダメージを増加させる」など）。

**カードの並べ替え方法:**
- **PC**: スピリットカードやモッドをクリックして、別のスロットにドラッグします。
- **コントローラー**: 十字キーまたはスティックでカードをハイライトし、決定ボタンで「持ち上げ」、目的のスロットに移動させます。

<Callout type="info" title="モッドの合成">
重複するモッドカードを入手すると、ゲームから合成のプロンプトが表示されます。これは通常レベルアップ画面で行われますが、Tab/デッキメニューでデッキを整理しておくことで、メイン武器で常に最大のレベル3ボーナスを得られるようになります。
</Callout>

| スロットタイプ | 最大容量 | 操作の相互作用 |
| :--- | :--- | :--- |
| **武器カード** | 4スロット | 主要なダメージ源。入れ替えや並べ替えが可能。 |
| **モッドカード** | 武器ごとに3つ | 武器に装着。合成によりレアリティとステータスが向上。 |
| **スペルスロット** | 1スロット | 特定の詩人（ダンテ、ウェルギリウスなど）に紐付け。 |

</Section>

<Section icon={Shield} title="最適化とトラブルシューティング">

2026年7月16日のアーリーアクセス開始時点では、入力遅延に影響するパフォーマンスの問題が発生する場合があります。**ヘル・メイデンの操作**とシステム設定を適切に構成することで、ランの後半分における生存率を大幅に向上させることができます。

**より良い操作のためのパフォーマンスのヒント:**
- **DirectX 11モード**: 入力ラグを感じる場合は、Steamの起動オプションでDX11を使用してください。
- **マウス感度**: 設定メニューでこれを調整し、照準レティクルが自身の反応速度に合ったスピードで動くようにしてください。
- **視覚的な明瞭さ**: 激しいウェーブ中に照準が困難になる場合は、不要な画面の揺れ（スクリーンシェイク）をオフにしてください。
