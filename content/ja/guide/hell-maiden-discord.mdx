<Section icon={Map} title="ヘル・メイデン Discord コミュニティへの参加">

公式の **ヘル・メイデン Discord** と繋がることは、進化し続けるこの地獄のようなサバイバル・デッキ構築ゲームの状況を把握するための最も効果的な方法です。AstralShift によって開発されたこのゲームは、ダンテ・アリギエーリの『神曲』から強いインスピレーションを得ており、コミュニティサーバーもそのテーマの深さを反映しています。カードの合成に関するヒントを探している場合でも、ウェルギリウスやオウィディウスが登場する伝承（ロア）について議論したい場合でも、Discord サーバーはあらゆるタイプの生存者に構造化された環境を提供します。

**ビデオハイライト：**
- 地獄の深淵におけるサバイバル・デッキ構築メカニクスのビジュアルショーケース。
- 忠実な案内人ウェルギリウスや、辺獄（リンボ）の詩人オウィディウスなどの伝説的な味方の紹介。
- 地獄の裁判官、ミノス女王の部下たちとの激しい戦闘の様子。
- ダンテの武器を強化するためのカード進化と合成システムのデモンストレーション。

<YouTubeEmbed videoId="90EaQ4buCC8" title="ヘル・メイデン | ゲームプレイトレーラー" />

サーバーは、初心者とベテランのデッキビルダーの両方に対応するように設計されています。参加することで、特定のカード進化のフレームデータや、伝説的な味方の正確な出現率など、ゲーム内では必ずしも入手できない知識の宝庫にアクセスできるようになります。

| チャンネル名 | 目的 | アクセスレベル |
| :--- | :--- | :--- |
| **#announcements** | アップデートやパッチに関する AstralShift からの公式ニュース | 公開 |
| **#dev-log** | ヘル・メイデンの開発の舞台裏 | 公開 |
| **#general-chat** | 日常的な会話やプレイヤー同士の交流のためのメインハブ | 認証済み |
| **#strategy-talk** | カードの相乗効果やサバイバルビルドに関する深い議論 | 認証済み |
| **#bug-reports** | 技術的な問題を報告するための専用スペース | 認証済み |

<Callout type="tip" title="プロのヒント">
#announcements チャンネルの通知を有効にして、期間限定のコミュニティイベントやサプライズのパッチノートリリースを見逃さないようにしましょう。
</Callout>

<Section icon={Target} title="サーバーのナビゲーションとロール階層">

**ヘル・メイデン Discord** に入ると、プレイヤーはボットのいない環境を維持するための認証プロセスに迎えられます。認証が完了すると、サーバー内での体験を定義するロール（役割）を選択できます。これらのロールは、多くの場合、あなたのプレイスタイルや地獄の各層での進行状況に対応しています。

<CardGrid cols={3}>
  <Card title="生存者（Survivor）ロール">
    - **標準アクセス**
    - 一般的な議論への参加
    - 基本的な戦略の共有
  </Card>
  <Card title="詩人（Poet）ロール">
    - **コンテンツクリエイター**
    - メディア共有チャンネルへのアクセス
    - ファンアート紹介の優先権
  </Card>
  <Card title="裁判官（Judge）ロール">
    - **コミュニティのベテラン**
    - ガイドの検証において信頼される存在
    - 実験的なフィードバックループへのアクセス
  </Card>
</CardGrid>

ロールシステムは単なる見せかけではありません。AstralShift の開発者がフィードバックを分類するのに役立ちます。例えば、高レベルのロールを持つプレイヤーは、ヘル・メイデンのパブリック版に実装される前に、メタの変化やカードのバランス調整について話し合うプライベートな「評議会（Council）」チャンネルに招待されることがよくあります。

| ロール特典 | 説明 | 要件 |
| :--- | :--- | :--- |
| **ユニークなカラー** | メンバーリストで専用の色により目立つことができる | ロールの割り当て |
| **優先フィードバック** | あなたの提案が開発レビューセッションで強調される | 積極的な参加 |
| **ベータアクセス** | 公式リリース前に新しいカードセットをテストできる可能性 | 応募 / 活動状況 |
| **カスタム絵文字** | Discord 全体でヘル・メイデンをテーマにした限定絵文字を使用可能 | サーバーメンバーシップ |

<Callout type="info" title="重要">
ロールの選択は通常、#welcome または #roles チャンネルにある自動リアクションメニューで行われます。説明をよく読んでから選択してください。
</Callout>

</Section>

<Section icon={Shield} title="コミュニティガイドラインと安全性">

すべての生存者にとって健全な環境を維持するために、**ヘル・メイデン Discord** では厳格なルールセットが適用されています。ゲームのダークなテーマを考慮し、モデレーターは議論が尊重され、ゲームプレイ体験に焦点を当てたものになるよう尽力しています。

<Steps>
  <Step num="1" title="認証">
    最初のキャプチャまたはリアクションベースの認証を完了し、あなたが人間の生存者であることを証明します。
  </Step>
  <Step num="2" title="ルールを読む">
    #rules チャンネルに移動します。ネタバレ、スパム、ハラスメントに関するポリシーを確認してください。
  </Step>
  <Step num="3" title="自己紹介">
    #introductions チャンネルへ向かい、他のプレイヤーと知り合い、お気に入りのデッキのアーキタイプを共有しましょう。
  </Step>
</Steps>

モデレーションチームは、AstralShift のスタッフと、ゲームのメカニクスに精通したコミュニティボランティアで構成されています。彼らは助けになりますが、ガイドラインが繰り返し無視された場合には、警告や追放を行う権限も持っています。

| 違反行為 | 初回違反 | 繰り返しの違反 |
| :--- | :--- | :--- |
| **スパム行為** | 口頭警告 | 一時的なミュート |
| **未表示のネタバレ** | メッセージの削除 | 24時間の禁止 |
| **ハラスメント** | 即時ミュート | 永久追放 |
| **自己宣伝** | 警告 / 削除 | サーバーからのキック |

<Callout type="warning" title="警告">
#spoiler-chat は、『神曲』を翻案したゲーム後半のストーリー展開について、タグなしで議論することが許可されている唯一の場所です。
</Callout>

</Section>

<Section icon={Swords} title="高度な戦略とメタ議論">

**ヘル・メイデン Discord** の最も価値のある側面の一つは、プレイヤーベースの集合知です。#strategy-talk チャンネルでは、プレイヤーがカードの進化経路の効率や、敵の「地獄の潮流（Hellish Tide）」に対抗するための最善の方法を分析しています。

コミュニティでは、頻繁に議論され洗練されているいくつかの「メタビルド」を分類しています：

| ビルドタイプ | コアメカニクス | 難易度 |
| :--- | :--- | :--- |
| **ウェルギリウスの盾** | 高い体力再生を伴う防御的な合成 | 初心者 |
| **オウィディウスの詩** | 素早いカードの回転とクールダウン短縮 | 上級者 |
| **ミノスの審判** | ハイリスク・ハイリターンの瞬間火力カード | エキスパート |
| **深淵の群れ** | 敵の群れを逸らすための味方の召喚 | 中級者 |

<CardGrid cols={2}>
  <Card title="デッキの最適化">
    - **カード合成**: 序盤は低ティアカードの組み合わせに集中する。
    - **リソース管理**: 「ソウルエッセンス」の数に注意を払う。
    - **進化のタイミング**: コストが増加する場合は、早すぎる進化を避ける。
  </Card>
  <Card title="生存戦術">
    - **カイティング**: 環境を利用して敵をまとめ、範囲攻撃（AoE）を当てる。
    - **味方との相乗効果**: ウェルギリウスのアクティブバフにデッキを合わせる。
    - **ボス対策**: ミノス女王のために高ダメージカードを温存する。
  </Card>
</CardGrid>

<Callout type="success" title="成功事例">
多くのプレイヤーが、Discord で共有された「オウィディウスの詩」ビルドに従うことで、標準的なビルドよりも 40% 効率的に地獄の第三圏をクリアできたと報告しています。
</Callout>

</Section>

<Section icon={Sparkles} title="開発者との交流とバグ報告">

AstralShift はコミュニティに対して非常に反応が良いことで知られています。**ヘル・メイデン Discord** には構造化されたバグ報告システムがあり、すべての技術的な不具合が記録され、対処されるようになっています。複雑なカードの相互作用に依存するゲームにとって、開発者へのこの直接的なラインは極めて重要です。

効果的にバグを報告するために、プレイヤーは #bug-reports チャンネルで以下の形式を使用することが推奨されています：

| フィールド | 必要な情報 |
| :--- | :--- |
| **プラットフォーム** | PC、Steam Deck、またはコンソール |
| **カードコンボ** | バグ発生時にどのアクティブカードを使用していたか |
| **説明** | 意図しない動作の簡単な要約 |
| **頻度** | 一度だけ起きたのか、再現可能か |

<Callout type="tip" title="プロのヒント">
短いビデオクリップやデッキレイアウトのスクリーンショットを添付すると、開発者が問題を診断するスピードが大幅に向上します。
</Callout>

バグ報告以外にも、「提案（Suggestions）」チャンネルはコミュニティの声が真に輝く場所です。2026年のアップデートで見られるカード進化の多くは、元々 Discord サーバー内で議論されたプレイヤーのコンセプトから生まれました。

</Section>
