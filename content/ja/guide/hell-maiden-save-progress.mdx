<Section icon={Sparkles} title="カードマージとメタデッキ戦略">

**ヘル・メイデンのセーブ進行状況**の鍵となるコンポーネントは、カードシステムをマスターすることです。カード自体はランごとにリセットされますが、どの組み合わせが最適かという知識があれば、後半のステージでの進行が速くなります。ゲームには5つの異なるModカードタイプがあり、マージを通じてレベル3まで進化させることができます。

| Modタイプ | 主な効果 | 戦略的用途 |
| :--- | :--- | :--- |
| **ブースト** | ステータス上昇 | 直接的な武器パワー |
| **エフェクト** | 状態異常 | 燃焼、鈍足、毒 |
| **スプレッド** | 複数武器のバフ | シナジービルド |
| **ミラクル** | 複数の強力なブースト | レア、高インパクト |
| **カース** | 強力なバフ / 重い代償 | リスクとリターンのバランス |

マージには、同じレベルとタイプのカードが2枚必要です。例えば、レベル1の「攻撃サイズ」Modを別のレベル1の「攻撃サイズ」Modにドラッグすると、ビジュアルとステータスが強化されたレベル2バージョンが作成されます。これは、ゲーム終盤のボスに対して効果的にダメージをスケールさせる唯一の方法です。

<Callout type="info" title="マージメカニズム">
デッキの変更はレベルアップ画面でのみ可能です。この時間を利用して、異なる武器間でModカードを入れ替え、シグネチャー能力を最適化してください。
</Callout>

</Section>
