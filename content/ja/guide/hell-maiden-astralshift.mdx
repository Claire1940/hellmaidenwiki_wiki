<Section icon={Sparkles} title="カード合成システムのマスター">

**ヘル・メイデン アストラルシフト**の最大の特徴は、プレイヤーのパワーの扱い方にあります。単純なステータスアップの代わりに、さまざまな攻撃、呪文、パッシブを表すカードを収集します。「合成（Merge）」システムでは、同一または互換性のあるカードを組み合わせて、独自の特性を持つ「進化（Evolved）」バージョンを作成できます。

<CardGrid cols={3}>
  <Card title="基本カード">
    - **高頻度**: 下級悪魔から頻繁にドロップ。
    - **低コスト**: 展開しやすく、サイクルを回しやすい。
    - **基礎**: すべての上位進化に不可欠。
  </Card>
  <Card title="合成カード">
    - **強化ステータス**: 射程、ダメージ、クールタイムが向上。
    - **相乗効果ボーナス**: 二次的な属性効果を得ることが多い。
    - **中盤の戦力**: 中盤のウェーブを生き抜くために必須。
  </Card>
  <Card title="進化カード">
    - **独自のメカニクス**: 攻撃を新しい形態へと変貌させる。
    - **画面一掃**: 敵の群れ全体を殲滅する能力。
    - **希少な条件**: 特定のカードの組み合わせが必要。
  </Card>
</CardGrid>

ハイレベルなプレイには、進化ルートを理解することが不可欠です。単にランダムにカードを選ぶのではなく、地獄の深層で見られる特定の脅威に対処できる特定の「アーキタイプ」を目指して構築する必要があります。

| カードティア | 入手しやすさ | アップグレード方法 | リソースコスト |
| :--- | :--- | :--- | :--- |
| **コモン** | 高（全レベル） | 通常ドロップ | 低 |
| **アンコモン** | 中 | コモン×2を合成 | 中 |
| **レア** | 低 | アンコモン×2を合成 | 高 |
| **エピック** | 極めて低 | 特定のレシピ | 極めて高 |
| **レジェンダリー** | ボス/イベント限定 | 進化の触媒 | 膨大 |

<Callout type="warning" title="戦略アラート">
早い段階でカードを合成しすぎないように注意してください。進化カードは強力ですが、2つの別々の基本カードが提供していた「攻撃範囲」を失うと、複数の方向から囲まれた際に脆弱になる可能性があります。
</Callout>

<Section icon={Map} title="協力者と物語の進行">

戦闘は孤独なものですが、**ヘル・メイデン アストラルシフト**の旅には、ガイドや道徳的な指針となる伝説的な人物たちが登場します。これらの協力者は単なる伝承以上の存在であり、失敗しそうなランを救うことができるパッシブバフや能動的な介入を提供してくれます。

**深淵の主要な協力者：**
- **ウェルギリウス**: 忠実なガイド。戦術的なアドバイスを提供し、ダンテが深淵の険しい地形を進むのを助けます。ゲームプレイ面では、ウェルギリウスはしばしばカード選択の「引き直し（Reroll）」の機会を与えてくれます。
- **オウィディウス**: 伝説的な辺獄（リンボ）の詩人。オウィディウスの存在は旅の哲学的・物語的な重みに焦点を当てており、しばしば「詩的なインスピレーション」を通じてランの間の永続的なアップグレードをアンロックします。

| 協力者 | 役割 | 主な利点 | 解放条件 |
| :--- | :--- | :--- | :--- |
| **ウェルギリウス** | ガイド | カード引き直し / ナビゲーション | 初期状態 |
| **オウィディウス** | 詩人 | 永続的なメタ進行 | 辺獄（リンボ）に到達 |
| **ベアトリーチェ** | インスピレーション | 体力回復 / バフ | 深層到達の節目 |

<Section icon={Shield} title="階層での生存：戦闘戦術">

**ヘル・メイデン アストラルシフト**における生存は、移動と管理にかかっています。「地獄の乙女」あるいはダンテとして、あなたは常に「ザ・マリス（悪意）」に追われます。これは、同じ場所に長く留まりすぎると移動を制限する忍び寄る力です。

<Steps>
  <Step num="1" title="初期の収集">
    小規模な敵のグループをカイト（引き撃ち）することからランを始めます。「魂の断片」を拾い、最初のカードを手に入れることに集中しましょう。可能であれば移動速度アップのカードを優先してください。
  </Step>
  <Step num="2" title="デッキの整理">
    10分経過するまでに、少なくとも2つの合成カードを持っているべきです。手札が詰まるのを防ぐため、意図した進化ルートに合わない新しい基本カードの取得は控えましょう。
  </Step>
  <Step num="3" title="ボスの準備">
    ミノス女王や他の階層の守護者に挑む前に、「単体ターゲット」へのバーストダメージカードがあることを確認してください。範囲攻撃はウェーブには最適ですが、ボスは集中砲火を必要とする膨大な体力を備えています。
  </Step>
  <Step num="4" title="最終決戦">
    最深層では、画面が弾幕で埋め尽くされます。シールドや短距離テレポートなどの「防御ユーティリティ」カードを使用して、弾幕パターンを切り抜けましょう。
  </Step>
</Steps>

このゲームのボスは単なる「弾丸のスポンジ」ではありません。彼らには明確なフェーズがあります。例えば、ミノス女王は「裁き」のメカニクスを使用し、床のエリアをマークします。これらのエリアに立っていると、即座にカードが破棄され、攻撃能力が大幅に弱体化します。

| ボス名 | 場所 | 主な攻撃 | 戦略 |
| :--- | :--- | :--- | :--- |
| **ミノス女王** | 裁きの階層 | ガベルスラム / カード破棄 | 移動を続け、単体攻撃に集中 |
| **マリス・コア** | 深淵の底 | 触手攻撃 / 暗黒 | 光属性のカードが必要 |
| **堕落した軍団** | 暴力の階層 | シールドチャージ / 方陣 | 貫通ダメージが不可欠 |

<Callout type="info" title="重要な注意点">
ヘル・メイデン アストラルシフトの難易度は、ランの前に装備する「罪（Sins）」の数によってスケールします。罪はゲームを難しくしますが、レアやエピックカードのドロップ率を大幅に向上させます。
</Callout>

</Section>
