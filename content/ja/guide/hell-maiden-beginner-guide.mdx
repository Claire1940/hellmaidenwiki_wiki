<Section icon={Zap} title="カード合成と修正カードのマスター">

ヘル・メイデンの体験において最もユニークな要素はカードシステムです。単にパークを選択するだけの伝統的なローグライクとは異なり、このゲームでは特定の武器に修正カードをスロットし、重複したカードを合成してレベルを上げることができます。各武器にはスロット数に限りがあるため、メインのダメージソースにどの効果を適用し、サブのサポート武器に何を充てるか戦略を立てる必要があります。

<Callout type="warning" title="リソース管理">
カードを合成すると、ベースとなるカードを消費して高レベル版を作成します。メイン武器の唯一の速度修正カードを合成する前に、予備のダメージ源を確保しているか確認してください。
</Callout>

| カード名 | 主な効果 | 最適なシナジー |
| :--- | :--- | :--- |
| **Sataris Path** | ダメージ+100% / 攻撃速度低下 | Swift Strikes |
| **Swift Strikes** | 攻撃速度の上昇 | Sataris Path |
| **Kindled Embers** | 発射体数の増加 | Venus's Favor |
| **Flare Up** | 30%の確率で爆発が発生 | 高ダメージビルド |
| **Eternal Prestige** | 5%の確率でHP吸収 | 攻撃速度の速い武器 |

ロードアウトを構築する際は、**修正カード（Modification Cards）**（通常は青または金）を探しましょう。これらは武器カードの間に配置することで、隣接する複数のアイテムに効果を適用できます。例えば、**ヴィーナスの加護（Venus's Favor）**は隣接する武器の発射体数を増やすため、焦熱の羽ペンやキンドルド・エンバーズを使用するビルドには「必須」のカードです。

  <Step num="1" title="重複カードを集める">

  <Step num="2" title="インベントリを開く">

    同じ種類・同じレベルのカードを別のカードの上にドラッグして、レベル2またはレベル3を作成します。
  </Step>
  <Step num="4" title="スロットの最適化">
    ステータスボーナスを最大化するため、最もレベルの高い修正カードを最も強力な武器の隣に配置します。
  </Step>
</Steps>

<Section icon={Shield} title="生存戦術と敵の行動パターン">

ヘル・メイデンにおいて、生存はダメージと同じくらい重要です。追い詰められると数秒でランを終わらせてしまう「自爆兵」や「エリート・スレイヤー」が頻繁に襲ってきます。敵の予兆を理解し、安全な距離を保つことがプロプレイヤーの証です。飛び道具を使う敵は、自分の攻撃エフェクトによる視覚的な混乱の中で弾が見えにくくなるため、特に危険です。

<Callout type="info" title="防御ステータス">
敵の群れからダメージを受けやすい場合は、**全般防御（General Defense）**と**近接耐性（Melee Resistance）**を優先してください。防御ステータスが13%以上あれば、ボス戦でのミスが許容される範囲が大幅に広がります。
</Callout>

**主要な生存戦略:**
- **カイティング（引き撃ち）**: 常にマップの中央を中心に大きな円を描くように移動し、敵をまとめて範囲攻撃（AOE）で処理します。
- **ダッシュ管理**: 敵に完全に囲まれる「ボディブロック」状態から脱出するためにダッシュを温存しておきましょう。
- **アルティメットのタイミング**: ダンテの**インフェルノ・アルティメット（Inferno Ultimate）**は、ダメージを与えるだけでなく、一瞬の無敵時間と画面の敵を一掃するために使用します。
- **経験値の回収**: 経験値ジェムを長時間放置しないでください。**磁石（Magnet）**のパワーアップや**フェイタル・イールド（Fatal Yield）**カードを使用して、常にレベル曲線の先を行くようにしましょう。

| 敵のタイプ | 危険度 | 最適な対策 |
| :--- | :--- | :--- |
| **キャタピー** | 低 | あらゆる範囲攻撃 |
| **自爆兵** | 高 | ダッシュで離脱、遠距離攻撃 |
| **遠距離インプ** | 中 | 飛び道具防御、絶え間ない移動 |
| **エリート・スレイヤー** | 極めて高 | アルティメット能力、カイティング |

</Section>

<Section icon={Swords} title="ボス攻略：ミノスの撃破">

辺獄（リンボ）のクライマックスは、冥界の女王**ミノス**との戦いです。この戦いは、あなたの立ち回りとビルドの質が試されます。ミノスはショットガンのような弾幕攻撃や、地面を走るレーザーなど、多彩な攻撃を繰り出します。焦熱の羽ペンはターゲットにある程度近づく必要があるため、彼女の強力な近接バーストを避けるために「ヒット・アンド・アウェイ」の戦術をマスターしなければなりません。

<Callout type="tip" title="ミノス攻略のコツ">
ミノスの正面ではなく、少し横に位置取るようにしましょう。彼女の弾幕攻撃のほとんどは、プレイヤーの現在位置に向かって円錐状に発射されます。
</Callout>

<Steps>
  <Step num="1" title="フェーズ1：弾幕">
    横方向に移動して緑色のショットガン弾を回避します。このフェーズでは彼女に向かってダッシュしないでください。
  </Step>
  <Step num="2" title="フェーズ2：レーザー">
    床のインジケーターに注意してください。ビームの間の隙間にすぐに移動します。
  </Step>
  <Step num="3" title="フェーズ3：ラスト・ライツ">
    ミノスは円状に高ダメージの弾幕を連続発射します。中距離を保つことで、隙間を見つけやすくなります。
  </Step>
  <Step num="4" title="フィニッシュ">
    彼女の体力が少なくなったら、**インフェルノ・アルティメット**を発動し、軽微なダメージを無視しながら残りのHPを焼き尽くしましょう。
  </Step>
</Steps>

| ミノスの攻撃 | ダメージタイプ | 回避方法 |
| :--- | :--- | :--- |
| **ショットガン弾** | 飛び道具 | 横方向への移動 |
| **ヘルファイア・レーザー** | 範囲 / 床 | 安全地帯に立つ |
| **雑魚召喚** | 近接 | 範囲攻撃/爆発を利用 |
| **ラスト・ライツ** | 重飛び道具 | 距離を保ち正確にカイト |

</Section>
